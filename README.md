# Blender Weapons Modelling
 
This project involves prototyping 3D Computer Graphics Models of 3 different melee weapons: a concept Electric-Sword, Darth Vaderâ€™s Lightsabre (from Star Wars), and Mjolnir (from the Thor movies). The models were created using Blender to gain practical experience in 3D modeling using Computer Graphics techniques.
